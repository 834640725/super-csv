Downloading Super CSV

Prerequisites

Java 1.5+
Super CSV is compiled for Java 1.5

Maven users

If you are using Maven, simply copy the following dependency into your pom.xml file. The artifact is hosted at Maven Central, and is standalone (no dependencies).

<dependency>
    <groupId>net.sf.supercsv</groupId>
    <artifactId>super-csv</artifactId>
    <version>2.4.0</version>
</dependency>

If you wish to use the Dozer extension (with deep mapping and index-based mapping support), then you will also need the following:

<dependency>
    <groupId>net.sf.supercsv</groupId>
    <artifactId>super-csv-dozer</artifactId>
    <version>2.4.0</version>
</dependency>

If you wish to use the Joda-Time extension (cell processors for parsing/formatting Joda-Time classes), then you will also need the following:

<dependency>
    <groupId>net.sf.supercsv</groupId>
    <artifactId>super-csv-joda</artifactId>
    <version>2.4.0</version>
</dependency>

If you wish to use the Java 8 extension (cell processors for parsing/formatting Java 8 classes), then you will also need the following:

<dependency>
    <groupId>net.sf.supercsv</groupId>
    <artifactId>super-csv-java8</artifactId>
    <version>2.4.0</version>
</dependency>

Everyone else
